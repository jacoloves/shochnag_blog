Dockerについてわかったこと

docker環境を構築してC言語の開発してるのだがほしいソフトをインストールする際に良くてまどうのでDckerらしいかは定かではないが私が覚えたインストールの仕方をメモする

Dockerfileにほしいソフトを記載する
Dockerfile

FROM ubuntu:18.04

#ここがapt-get updatae and apt-get installの場所
RUN apt-get update && \
    apt-get install -y \
    #gitをインストールしている
    git \
    #cコンパイル用にbuild-essentialをインストール
    build-essential \
    #makefileを使いたいのでインストール
    cmake \
    #これはなんだっけ？
    libssl-dev \
    #buildツールが使いたかったのでインストール
    gdb \
    #vimを使いたかったのでインストール
    vim
#いちいちdockerに入るたびにcdするのが面倒なので作業用ファイルに移動するようにする
WORKDIR /cpp-dev

docker-compose buildコマンドでコンテナに設定を反映する。
docker-compose

docker-compose build

ここが参考になった
いいかんじに色々と書いてある
